# Portfólio da Mariana

Portfólio profissional de **Mariana Viana** — Platform & DevEx Engineer.
Site de página única (SPA) construído com React, Vite e Tailwind CSS.

🔗 **Deploy:** https://github.com/rruandev/portifolio-da-mariana

## Tecnologias

- React 18 + TypeScript
- Vite 6
- Tailwind CSS 4
- Motion (animações)
- Radix UI / shadcn

## Como rodar localmente

```bash
npm install      # instala as dependências
npm run dev      # inicia o servidor de desenvolvimento (http://localhost:5173)
```

## Build de produção

```bash
npm run build    # gera os arquivos otimizados em dist/
npm run preview  # visualiza o build localmente
```

## Estrutura

```
src/
  app/
    components/
      portfolio/   # seções: Hero, Experience, Projects, Qualifications, Contact, Navbar
      ui/          # componentes de interface reutilizáveis
```
